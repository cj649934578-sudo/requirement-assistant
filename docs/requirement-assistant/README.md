# Requirement Assistant Docs

This directory stores workflow documents generated by the `requirement-assistant` skill.

## Directories

- `intent/`: `YYYY-MM-DD-<topic>-intent-summary.md`
- `plan/`: `YYYY-MM-DD-<topic>-execution-plan.md`
- `status/`: `YYYY-MM-DD-<topic>-run-status.md`

## Notes

- Use the same `<topic>` slug across the same workflow run when possible.
- `run-status` is typically updated in place for the active run.
- These files support pause/resume execution and make the current workflow phase explicit.
